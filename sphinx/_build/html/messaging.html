The biogeme.messaging module is a logger to manage the verbosity of the package.

screenLevel (int) –

level of message that must be displayed on the screen:

  • 0: no output (default)

  • 1: warnings only

  • 2: only warnings and general information

  • 3: more verbose

  • 4: debug messages
